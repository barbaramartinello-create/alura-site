# 🚀 NovaTech

Site moderno e responsivo desenvolvido com HTML, CSS e JavaScript.

Projeto criado para apresentar uma agência digital fictícia chamada **NovaTech**, focada em tecnologia, design e inovação.

---

## ✨ Funcionalidades

- ✅ Layout moderno
- ✅ Design responsivo
- ✅ Navegação suave
- ✅ Cards animados
- ✅ Formulário de contato
- ✅ Header com efeito ao rolar
- ✅ Interface profissional

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts

---

## 📁 Estrutura do Projeto

```bash
📦 novatech
 ┣ 📜 index.html
 ┣ 📜 style.css
 ┣ 📜 script.js
 ┗ 📜 README.md
```

## 💻 Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/novatech.git
```

2. Abra a pasta do projeto

3. Execute o arquivo:

```bash
index.html
```
